# CodeQuest Academy

Application statique d’apprentissage HTML/PHP, compatible avec GitHub Pages.

## Publier sur GitHub Pages

1. Créez un dépôt GitHub et ajoutez `index.html` à la racine.
2. Ouvrez **Settings → Pages** dans le dépôt.
3. Dans **Build and deployment**, choisissez **Deploy from a branch**.
4. Sélectionnez la branche `main`, le dossier `/ (root)`, puis **Save**.
5. GitHub affichera l’adresse publique après le premier déploiement.

## Sauvegarde

La progression, les brouillons et le mode de navigation sont enregistrés dans le `localStorage` du navigateur. Ils persistent après fermeture de la page, mais ne sont pas automatiquement synchronisés entre un ordinateur et un téléphone.

## Utilisation mobile

La barre inférieure permet de passer entre **Cours**, **Code** et **Résultat**. Quand le clavier virtuel est ouvert, l’aperçu est temporairement masqué pour laisser le maximum de place à l’éditeur.
